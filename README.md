**English**
- ✅ **188+ channels** — Telewebion national & provincial channels + Persian networks
- 🖼️ **Real logos** pulled straight from Telewebion's official source
- 📅 **EPG (program guide)** for major channels, with matching logos

## 🚀 How to Use / نحوه استفاده

### TiviMate / IM Player / OTT Navigator

1. Open your player → **Add Playlist** → **M3U Playlist URL**
2. Paste the **Playlist** link above
3. Go to **Settings → EPG** and paste the **EPG** link
4. Refresh and enjoy 🎉

**فارسی:**
1. در پلیر خود → **افزودن پلی‌لیست** → **آدرس M3U**
2. لینک **Playlist** بالا را وارد کنید
3. به بخش **تنظیمات → EPG** بروید و لینک **EPG** را وارد کنید
4. به‌روزرسانی کنید و لذت ببرید 🎉

## ⚙️ How It Works / این پروژه چطور کار می‌کند

The playlist is rebuilt automatically by a GitHub Actions workflow:

1. Fetches the latest channel list directly from Telewebion's API
2. Tests every stream link and keeps only the live ones
3. Attaches logos, Persian names, and EPG IDs
4. Commits the updated `final.m3u` back to the repository

پلی‌لیست به‌صورت خودکار توسط GitHub Actions ساخته می‌شود: لیست کانال‌ها از API تلوبیون گرفته می‌شود، هر لینک تست می‌شود، و فقط کانال‌های زنده با لوگو و اطلاعات برنامه نگه داشته می‌شوند.

## ⚠️ Notes / نکات

- This project only **aggregates publicly available free-to-air streams**. It hosts no content.
- Some channels may be geo-restricted to Iran; a VPN may be required outside Iran.
- این پروژه فقط **لینک‌های پخش آزاد و عمومی** را گردآوری می‌کند و هیچ محتوایی میزبانی نمی‌کند.
- برخی شبکه‌ها ممکن است فقط داخل ایران در دسترس باشند؛ خارج از ایران ممکن است به VPN نیاز باشد.
